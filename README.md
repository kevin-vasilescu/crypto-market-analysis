# Cryptocurrency Market Analysis

A quick data analysis project examining crypto market trends, volatility patterns, and price movements.

## Overview

This project analyzes cryptocurrency price data to identify trends and patterns in market behavior. Uses Python with pandas and matplotlib for data processing and visualization.

## Features

- **Data Collection**: Fetches historical crypto price data
- **Trend Analysis**: Identifies bullish/bearish patterns
- **Volatility Metrics**: Calculates standard deviation and rolling averages
- **Visualizations**: Charts and graphs for market insights

## Quick Start

```bash
pip install pandas matplotlib requests
python analysis.py
```

## Dataset

Uses free crypto market data from public APIs (CoinGecko, etc.)

## Results

Key findings include price correlations, volatility spikes, and seasonal trends across major cryptocurrencies (BTC, ETH, etc.).
